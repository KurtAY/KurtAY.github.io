Coding Challenge 1 This project was one where I just displayed a message in the various ways you can display text in a webpage. In an alert box, on the actual web document and in the console.
Coding Challenge 2 This was a series of execises given on the first day. It incooperated the use of strings fucntions and variables, my first time using these in JavaScript.
Strings 1 The first exercise I did involving strings. I created a variable with a string message, displayed it, then created converted it to uppercase and displayed the uppercase version below.
Strings 2 I simply added a string number to an integer number to see the result. The result was the string and the int printed out next to one another. No mathematics was performed.
Arrays 1/Strings 3 For this, I created an array containing 3 strings and outputted it. I then used the push method to add another string to the array and printed it. Lastly, I used the pop method to remove the last string I added to return to the original array.
Conditionals 1 Used if statments to check if a person object I created is within the suitable age (20-40). Verification is printed along with age and if they are accepted or not.
Iteration 1 Here I created a simple for loop thay increments by 1 from 1 to 10, outputting each number after each iteration.
Iteration 2 This was similar to 'Iteration 1', however I only outputted the multiples of 2 from 1 to 10. I did this using an additional if statement and the modulo operator.
Iteration 3 - Fizz Buzz Create a method that prints the numbers 1 to 100 via a for loop, however if the number it’s outputting is divisible by 3 then output “Fizz” instead, if the number is divisible by 5 output “Buzz” instead. If the number is divisible by both 3 and 5, output “FizzBuzz”.
Iteration 4 Create a method accepts a number then, outputting what it did at each iteration, and how many iterations it took to get to the end. • If the number is divisible by 3, divide it by 3. • If it’s not, either add 1 or subtract 1 (to make it divisible by 3), then divide it by 3. • It stops when the number it reaches is 1
Strings 4 We’ll say that a “triple” in a string is a character appearing three times in a row. Create a function that accepts a string and returns the amount of times a triple exists in that string.
Dom 1 Create a function that creates a paragraph tag Create another function that changes the text of that paragraph tag to what is in a textbox Create another function that deletes the paragraph tag All of these functions should be called with a button(s)
JSON 1 Access a JSON file. Once loaded, display the objects retrieved in an organised fashion.
Garage Create a virtual garage with the following functionality; • Check in cars to the garage • Check out cars from the garage • Output the contents of the garage • Calculate the bill for a car, dependant on its attributes
